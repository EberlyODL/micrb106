The genetic code then is made up of a series of 4 bases arranged into sets of 3 letters. That makes for 64 unique 3 letter combinations. However, there are only 20 commonly used amino acids used. There are then several codes that designate each amino acid. The following chart details all the codons and their corresponding amino acid.

Some of these codons are unique. The one in green is called the start codon (AUG). Three other codons, the ones in red, are stop codons (UAA, UAG, UGA). By finding the start and stop codons it’s possible to find the exact sequence of amino acids used to construct a specific protein.
