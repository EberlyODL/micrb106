The lymphatic system is described as a system for collecting excess fluids as they accumulate in the tissues. That means that anything that is present in those fluids also gets collected, including any invaders that penetrate the skin. This makes the lymphatic system a good place for cells of the immune system to hang out and attack any invaders that get into the body.

Related Organs

Since the lymphatic system is a place for invaders to collect, many organs that are related to immunity or the immune system also get included as part of the lymphatic system. There are a number of organs that are related to this immunity function of the lymphatic system:

  • Red bone marrow: This is the site of all blood cell production including immune system cells.
  • Thymus: This is the site of T-cell maturation.
  • Lymph nodes and smaller lymph nodules: Lymph filters through these structures and immune system cells screen for invaders.
  • Spleen: Similar function as lymph nodes except that blood that is screened for invaders.
